Supervised Learning and Unsupervised Learing

　　Supervised Learning:
　　in supervised leanring ,we are give a data set and already know what our correct output should look like,having the idea that there is a relationship between the input and output.
　　Supervised learing problems are categorized into”regretion” and “classification” problems.in a regretion problem,we try to map input variables to some continous function.in a classfication problem,we are trying to map input variables into discrete categories.
　　Example 1:Given data about the size of houses on the real estate market, try to predict their price. Price as a function of size is a continuous output, so this is a regression problem.
　　Example 2:We could turn example 1 into a classification problem by instead making our output about whether the house “sells for more or less than the asking price.” Here we are classifying the houses based on price into two discrete categories, so this is a classfication problem.
　　Unsupervised Learning:
　　Unsupervised learning allows us to approach problems with little or no idea what our results should look like.we can derive structure by clustering the date based on relationships among the variables in the data.with the unsupervised learing there is no feadback based on the prediction results.
　　Example 1:Clustering: Take a collection of 1,000,000 different genes, and find a way to automatically group these genes into groups that are somehow similar or related by different variables, such as lifespan, location, roles, and so on.
　　Example 2:Non-clustering: The “Cocktail Party Algorithm”, allows you to find structure in a chaotic environment.(i.e. identifying individual voices and music from a mesh of sounds at a cocktail party).
